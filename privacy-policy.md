# Privacy Policy for Read It Later

**Last updated: April 23, 2026**

## Overview

Read It Later is a browser extension that lets you save web pages to read later. This privacy policy explains how the extension handles your data.

## Data Collection

This extension does **not** collect, transmit, or share any personal data with any third party.

## Data Storage

All data (page titles, URLs, favicons, and read/unread status) is stored **locally on your device** using Chrome's built-in storage API (`chrome.storage.local`). No data is ever sent to any external server.

## Permissions Used

| Permission | Purpose |
|---|---|
| `storage` | Save and manage your reading list locally on your device. |
| `tabs` | Retrieve the current tab's title, URL, and favicon when you click "Save this page", and open saved items in a new tab. |
| `activeTab` | Access the active tab's information only when you explicitly click the extension icon. |

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

## Changes to This Policy

If this policy is updated, the changes will be reflected in this document with an updated date.
